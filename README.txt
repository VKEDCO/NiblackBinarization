
- NiblackBinarization.rar contains an Eclipse Anroid 4.2 project
  that binarizes bitmaps with Niblack's method. 

  To run: 1) create /mnt/sdcard/input_images/ and place an image in there;
  2) run the project;
  3) the output binarized image will be placed into
     /mnt/sdcard/output_images/
  
  These directories can be modified in NiblackBinarizationAct.java.
